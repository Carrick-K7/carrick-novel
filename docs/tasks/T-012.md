# T-012: Novel 章节切换添加动画提示

## 目标
优化 Novel 阅读页，章节切换时添加动画提示，不要静默切换

## 验收标准
- [ ] 点击上一章/下一章时，显示切换动画
- [ ] 动画提示：如"正在加载下一章"或章节标题过渡动画
- [ ] 从目录点击进入章节时也有提示
- [ ] 动画流畅，不卡顿
- [ ] 移动端和桌面端都适用

## 动画方案（可选）
- 方案A: 淡入淡出过渡
- 方案B: 滑动过渡
- 方案C: 章节标题闪动提示
- 方案D: 加载指示器
