# T-008: Novel T-006 后续修复优化

## 背景
T-006 部署后发现多处渲染问题，需要后续修复优化。

## 修复内容

### 1. 渲染规则扩展
- 支持 `**【场景：...】**` 格式（加粗方括号）
- 支持 `## 场景X：...` 格式（二级标题）
- 支持 `**【第X章完】**` 格式
- 支持 `**（第X章 完）**` 格式
- 支持 `【第X章·完】` 格式

### 2. 渲染顺序修复
- 问题：先转 `**粗体**` 再匹配场景 → 匹配失败
- 修复：先匹配场景/结尾 → 再转粗体

### 3. 换行优化
- 移除分割线 `<hr>` 后的换行
- 移除场景 div 后的换行
- 移除章节结束 div 后的换行
