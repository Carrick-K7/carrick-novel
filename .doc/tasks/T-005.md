# T-005: Novel 切换章节自动跳转顶部

## 目标
修复 Novel 阅读页，切换章节时自动跳转到章节开头

## 验收标准
- [ ] 点击"上一章"/"下一章"按钮后，自动滚动到页面顶部
- [ ] 从目录页点击进入章节后，自动滚动到顶部
- [ ] 跳转动画平滑（可选）
- [ ] 移动端和桌面端都生效

## 技术方案
在 Read.vue 中：
- 监听 chapterIndex 变化
- 使用 `window.scrollTo(0, 0)` 或 `element.scrollIntoView()`
- 在 `onMounted` 或 `watch(chapterIndex)` 中执行
