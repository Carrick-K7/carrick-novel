## 问题分析

### 1. 章节动画问题
当前 `Read.vue` 中的动画 CSS：
```css
.chapter-fade-enter-from { opacity: 0; transform: translateX(20px); }
.chapter-fade-leave-to { opacity: 0; transform: translateX(-20px); }
```
需要移除 `transform: translateX`，只保留 `opacity` 淡入淡出。

### 2. 抽屉失效问题
- CDrawer 组件已正确导出并构建
- 可能是 CSS 变量问题或定位问题
- 需要检查抽屉在 PC 端的显示情况

### 3. PC 端重构
- 当前布局是移动端优先设计（底部导航、全屏抽屉）
- PC 端应该有更宽的布局、侧边栏、桌面级导航
- 需要添加 `@media (min-width: 768px)` 样式
