## 需求背景

当前所有内容在一个页面，需要独立的页面路由。

## 目标

设计并实现以下路由：

```
/                  # 首页 - 书籍列表
/book/:id          # 书籍详情 - 章节列表
/read/:id/:chapter # 阅读页面 - 具体章节
```

## 任务清单

- [ ] 安装 Vue Router
- [ ] 设计路由结构
- [ ] 创建首页组件（书籍列表）
- [ ] 创建书籍详情页
- [ ] 创建阅读页面
- [ ] 阅读进度同步到新路由
- [ ] URL 分享功能

## 验收标准

1. 每个页面有独立 URL
2. 刷新后保持阅读位置
3. 可直接分享链接到指定章节
