# T-006: Novel 记住章节上次浏览位置

## 目标
实现阅读进度记忆功能，用户返回已读章节时自动恢复到上次浏览位置

## 验收标准
- [ ] 阅读时自动记录当前滚动位置（每30秒或滚动停止时）
- [ ] 返回已读章节时自动恢复到上次位置
- [ ] 显示"上次阅读到这里"提示（可选）
- [ ] 支持手动回到顶部按钮
- [ ] 移动端和桌面端都生效
- [ ] 数据存储在 localStorage

## 技术方案
- 使用 localStorage 存储每个章节的滚动位置
- Key: `scroll_position_${bookId}_${chapterIndex}`
- Value: `{ scrollY: number, timestamp: number }`
- 监听 scroll 事件（节流）
- 章节加载时检查并恢复位置
