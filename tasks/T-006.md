# T-006: Novel 阅读体验优化

## 需求

### 1. 加载动画
- 页面刷新后显示加载动画，而不是先显示旧章节再突然显示新章节
- 左右翻页时也显示加载动画
- 加载完成后再渲染内容

### 2. 渲染样式（初音色 #39c5bb）
- `---` 三个横线渲染为初音色分割线
- `场景：` 开头的段落渲染为初音色
- `章节结束` 段落渲染为初音色

### 3. 自动播放功能
- 在底部功能区增加自动播放按钮
- 支持调节翻页速度（慢/中/快）
- 到达底部后，3秒内自动进入下一章
- 可随时暂停/停止

## 技术方案

### 加载动画
- 在 Read.vue 中添加 `isLoading` 状态
- 切换章节时设置 `isLoading = true`
- 内容加载完成后设置 `isLoading = false`
- 使用 CSS 动画显示加载效果

### 渲染样式
- 在渲染 Markdown 内容时，添加自定义解析
- 检测 `---` 替换为 `<hr class="miku-divider">`
- 检测 `场景：` 添加 `<span class="miku-scene">`
- 检测 `章节结束` 添加 `<div class="miku-end">`

### 自动播放
- 添加 `isAutoPlay` 状态
- 添加 `autoPlaySpeed` 配置（默认中速）
- 监听滚动到底部事件
- 底部停留3秒后自动调用 `nextChapter()`
- 底部添加控制栏：播放/暂停按钮 + 速度选择
