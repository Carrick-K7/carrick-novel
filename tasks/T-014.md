# T-014: Novel 支持左右滑动和上拉加载切换章节

## 目标
优化 Novel 阅读页章节切换交互

## 验收标准
- [ ] 支持左右滑动切换章节（左滑下一章，右滑上一章）
- [ ] 章节结尾支持上拉加载下一章
- [ ] 滑动/上拉时有视觉反馈
- [ ] 移动端和桌面端都适用

## 技术方案
- 使用 Touch events 检测滑动
- 使用 Pull-to-refresh 或自定义上拉加载
- 添加滑动动画和提示
