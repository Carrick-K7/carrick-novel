# T-013: Novel 阅读页置顶按钮自动隐藏

## 目标
优化 Novel 阅读页置顶按钮交互：页面停止滚动2秒后，按钮淡出消失

## 验收标准
- [ ] 向下滚动时置顶按钮显示
- [ ] 页面停止滚动2秒后，按钮淡出消失
- [ ] 再次滚动时按钮重新显示
- [ ] 动画流畅自然

## 技术方案
- 使用 setTimeout 检测滚动停止
- 滚动时清除 timer，停止滚动2秒后触发隐藏
- 使用 CSS transition 实现淡入淡出
