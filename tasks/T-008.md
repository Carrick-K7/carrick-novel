# T-008: Novel 正文页回到顶部按钮优化

## 目标
优化 Novel 阅读页"回到顶部"按钮的位置和交互

## 验收标准
- [ ] 按钮位置：页面右下角（fixed定位）
- [ ] 不在底部工具栏显示
- [ ] 交互逻辑合理（显示/隐藏时机）
- [ ] 平滑滚动到顶部
- [ ] 移动端和桌面端都适用

## 交互设计（建议）
- 默认隐藏
- 向下滚动一定距离后显示（如滚动超过 300px）
- 向上滚动或回到顶部后隐藏
- 可添加淡入淡出动画

## 技术方案
- 使用 fixed 定位：position: fixed; bottom: 20px; right: 20px;
- 监听 scroll 事件判断显示/隐藏
- 使用 opacity/visibility 实现过渡动画
