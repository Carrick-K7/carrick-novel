# T-011: Novel 阅读页交互优化

## 需求来源
Carrick 反馈当前阅读页交互体验需要优化

## 功能实现

### 1. 屏蔽双击放大
- 禁用阅读页的双击放大功能
- 防止误触导致页面缩放

### 2. 字号调整改用滑块
- 移除现有的 "A- / A+" 按钮
- 改用滑动条 (slider) 控制字号
- 范围保持 12px - 24px
- 实时预览字号变化

### 3. 上拉抽屉优化
- 当前：抽屉从底部弹出，遮挡工具栏
- 优化：抽屉高度减少，不遮挡顶部工具栏
- 效果：可以平滑切换到其他工具（目录、自动播放等）
- 或者改为从侧边滑出？（需确认）

## 技术实现

### 文件位置
- `projects/carrick-novel/src/views/ReaderView.vue`
- `projects/carrick-novel/src/components/ReaderToolbar.vue`

### 关键修改点
1. 添加 CSS `touch-action: manipulation` 屏蔽双击
2. 替换 `<input type="range">` 滑块组件
3. 调整抽屉组件的 `bottom` 定位和高度
