# T-011: Novel 阅读页交互优化

## 需求来源
Carrick 反馈当前阅读页交互体验需要优化

## 功能实现

### 1. 屏蔽双击放大 ✅
- 禁用阅读页的双击放大功能
- 防止误触导致页面缩放
- **实现**: 在 `.chapter-content` 样式中添加 `touch-action: manipulation; -webkit-touch-callout: none;`

### 2. 字号调整改用滑块 ✅
- 移除现有的 "A- / A+" 按钮
- 改用滑动条 (slider) 控制字号
- 范围保持 12px - 24px
- 实时预览字号变化
- **实现**: 
  - 替换为 `<input type="range" class="fontsize-range" v-model.number="fontSize">`
  - 添加 `onFontSizeChange()` 方法实时保存
  - 使用 Miku 色 `#39c5bb` 作为滑块拇指颜色

### 3. 上拉抽屉优化 ✅
- 当前：抽屉从底部弹出，遮挡工具栏
- 优化：抽屉高度减少，不遮挡顶部工具栏
- 效果：可以平滑切换到其他工具（目录、自动播放等）
- **实现**: 调整 `.bottom-drawer` 的 `max-height` 从 `70vh` 改为 `calc(100vh - 60px)`
