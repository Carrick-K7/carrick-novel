# T-007: Novel 底部工具栏 UI 重构

## 需求拆分

### 1. 无边框按钮 + 初色图标
- 移除按钮边框
- 图标颜色改为初色 (#39c5bb)
- 保持悬停/点击效果

### 2. 上拉抽屉配置
- 目录功能 → 上拉抽屉显示
- 自动播放 → 上拉抽屉显示速度和播放控制
- 字号调节 → 新增上拉抽屉

### 3. 字号调节功能
- 底部工具栏增加字号调节按钮
- 支持 A-/A+ 调节
- 移除工具栏显示/隐藏时的自动字号调整

## 技术方案

### 按钮样式
```css
.nav-btn {
  border: none;  /* 移除边框 */
  color: #39c5bb;  /* 初色图标 */
  background: transparent;
}
```

### 上拉抽屉组件
- 使用 Drawer/BottomSheet 组件
- 从底部滑出
- 点击外部关闭

### 字号调节
- 添加 fontSize ref (默认 16px)
- A-: fontSize -= 2
- A+: fontSize += 2
- 范围: 12px - 24px
- 保存到 localStorage
