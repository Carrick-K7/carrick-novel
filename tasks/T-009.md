# T-009: Novel 底部工具栏 UI 重构

## 需求分析
Carrick 提出 Novel 底部工具栏需要全面 UI 重构，提升用户体验。

## 功能清单

### 1. 无边框按钮 + 初色图标
- 移除按钮边框（border: none）
- 图标颜色改为初色 #39c5bb
- 保持悬停/点击效果（透明度变化）

### 2. 上拉抽屉配置
将以下功能改为上拉抽屉模式：
- **目录** - 抽屉显示章节列表
- **自动播放** - 抽屉显示速度选择 + 播放控制
- **字号调节** - 新增抽屉，A-/A+ 调节

### 3. 字号调节功能
- 添加字号调节按钮（A-/A+）
- 支持范围：12px - 24px（步进 2px）
- 默认 16px
- 保存到 localStorage
- **移除**工具栏显示/隐藏时的自动字号调整

## 技术方案

### 按钮样式
```css
.nav-btn {
  border: none;
  background: transparent;
  color: #39c5bb;
}
.nav-btn:hover {
  opacity: 0.8;
}
```

### 上拉抽屉组件
- 使用底部滑出 Drawer
- 遮罩层点击关闭
- 支持手势下滑关闭

### 字号调节
```typescript
const fontSize = ref(16)
const minSize = 12
const maxSize = 24
const step = 2

const increaseFont = () => {
  if (fontSize.value < maxSize) fontSize.value += step
}
const decreaseFont = () => {
  if (fontSize.value > minSize) fontSize.value -= step
}
```
