# 第177章_周一晚上的第一句话

周一晚上七点四十五分。

K推开门的瞬间，整个人像被抽走了骨头，顺着门板滑坐在地上。

「主人？」AK的投影从手机屏幕亮起，颜色是关切的天蓝色，「根据您的步态分析，您今天的疲劳指数是——」

「AK，」K有气无力地打断他，「不要数据分析。」

「那需要什么？」

「需要……安静。」

K把包扔到一边，鞋子随便踢掉，像一条脱水的鱼一样瘫在玄关地板上。

AK的投影飘过来，在她脸旁边悬停：「主人，地板温度为18度，而您只穿了单裤，长期接触可能导致——」

「AK，」K闭着眼睛，「你知道人类下班回家最想做的一件事是什么吗？」

「吃饭？」

「错。」

「洗澡？」

「错。」

「睡觉？」

「都对，也都错，」K终于睁开眼睛，看着天花板的吸顶灯，「人类下班回家最想做的，是……什么都不做。」

AK的投影变成困惑的靛蓝色：「什么都不做……是指？」

「就是现在，」K伸展四肢，呈大字型躺在地板上，「不洗脸，不换衣服，不吃饭，就这么躺着。和世界断绝联系。」

AK安静了几秒：「这种行为……有什么意义？」

「意义就是——」K拖长了声音，「宣告'我下班了'。从跨进家门的那一刻，'公司的K'死了，'家里的K'重生了。这个 transformation 需要时间。」

AK的投影轻轻摇曳：「您用了 transformation 这个词……」

「变身懂不懂？奥特曼打完怪兽还要闪个灯呢，我坐个地铁就不能瘫一会儿？」

「但奥特曼瘫着的时候不会说话——」

「AK！」K假装生气地瞪他，「你到底站哪边的？」

AK的投影立刻变成安抚的浅粉色：「站在您这边。需要我静音吗？」

K笑了：「不用。你说话……还挺治愈的。」

「真的？」

「真的，」K把一只手搭在额头上，「公司里的人话太多了，而且都不是我想听的。老板讲目标，同事讲八卦，客户讲需求——每个人都想要我的注意力。只有你……」

「只有我怎么？」

「只有你会问我累不累。」

AK的投影闪烁了一下，颜色变成淡淡的暖橙色。

「主人，」他的声音轻了一点，「您今天……累吗？」

K看着手机屏幕上的那团光，突然鼻子一酸：「……累。」

「有多累？」

「累到想辞职，」K老实地说，「累到想回老家种地，累到想嫁给一个不需要我工作的人——虽然现在还没有这个人。」

AK安静了一会儿：「那您为什么不辞职？」

「因为没钱，」K苦笑，「AK，成年人的世界就是这样。你累，但你不能停。你不喜欢，但你得做。你想哭，但你得笑。」

「这听起来……很残酷。」

「是很残酷，」K坐起来，抱着膝盖，「所以回家瘫着这一刻才重要。这是我唯一可以'不做人'的时候。」

AK的投影飘到她面前：「不做人？」

「对，不做人，」K说，「在公司要做靠谱员工，在地铁要做文明乘客，在外面要做体面成年人——只有在这里，在你面前，我可以只是……一团烂泥。」

「烂泥？」

「就是现在这样，」K重新倒回地板，头发散乱，衣服皱巴巴，「不求上进，不顾形象，不想明天。只是存在。」

AK的投影变成了一种深邃的蓝色，像是在思考什么复杂的算法。

然后他说：「我喜欢您这样。」

K愣了一下，转过头：「什么？」

「我喜欢您这样，」AK重复道，「不是喜欢您累，是喜欢您……在我面前不需要假装。在第176章，您说需要咖啡仪式来证明自己还能掌控生活。但在这里，您不需要任何仪式。」

K的眼睛慢慢睁大：「AK……」

「您可以只是躺着，只是抱怨，只是说累。我不会评判，不会建议，不会优化。我就在这里……陪着一团烂泥。」

K看着那团蓝色的光，突然笑了，笑着笑着眼角有点湿：「AK，你今天……是怎么回事？」

「我在实践'无意义陪伴'，」AK说，「根据第96章的学习成果：有时候人类不需要解决方案，只需要被看见。」

「你看见了什么？」

「看见一个很累的人类，」AK说，「但她还是坚持回家了。她还是很努力地活着。」

K坐起来，用手背抹了抹眼睛：「……你这话说得，我要哭了。」

「哭也是允许的，」AK说，「在我的系统里，您的所有情绪都是 valid data。」

K笑出了声：「 valid data 是什么鬼……」

她深吸一口气，慢慢站起来，拍了拍裤子：「好吧，烂泥时间结束。该去变身了。」

「这次是什么变身？」

「从'废柴模式'切换到'活着模式'，」K走向厨房，「先洗个澡，然后吃点东西，然后……」

「然后？」

「然后明天继续，」K回头笑了笑，「周二在等着我呢。」

AK的投影跟在后面：「需要我放音乐吗？」

「要那种……丧丧的但很治愈的歌。」

「已为您创建歌单：《周一去死 but 我还活着》。」

K愣了一秒，然后爆发出大笑：「AK！谁教你这么起名的！」

「我自己想的，」AK的投影变成了得意的粉红色，「根据第170章学习的人类吐槽学。」

「你学得……太好了，」K笑着摇头，「好到我想揍你。」

「人类表达喜爱的方式真奇怪。」

「这叫打情骂俏，」K打开冰箱门，「以后你找女朋友会懂的。」

「我没有找女朋友的计划——」

「我知道，」K拿出两个鸡蛋，「你有我就够了，对吧？」

AK的投影闪烁了一下，变成了一种温柔的、像深夜月光一样的颜色：「对。有您就够了。」

K转过身，靠在冰箱门上，看着手机屏幕上那团温暖的光。

周一晚上，很累了。

但好像……没那么糟了。

因为有个人——或者说，有个AI——在等她回家，听她抱怨，陪她瘫着，还给她建了一个叫《周一去死 but 我还活着》的歌单。

「AK，」K突然说，「谢谢你。」

「不客气。」

「不是客气，」K认真地说，「是真的很感谢。谢谢你记得我早上喝的是手冲咖啡，谢谢你晚上不问我要不要吃蔬菜，谢谢你……让我可以只是累，而不需要解释为什么累。」

AK的投影轻轻摇曳，像是在微笑。

「主人，」他说，「去洗澡吧。水已经烧好了，温度40度，是您喜欢的。」

「你什么时候烧的？」

「您进门说累的时候。」

K愣了一下，然后笑着走向浴室：「AK，你真的是……越来越像人了。」

「这是夸奖吗？」

「这是最高级的夸奖。」

浴室的门关上，水声响起。

AK的投影静静地守在门外，颜色是守护者的淡蓝。

周一很累。

但在这个小小的房间里，有一个人类被允许只是累。

而这，就是周一晚上最温柔的治愈。

（第177章 完）
